Simplificando as declarações

Através da propriedade "flex" podemos simplificar as três declarações "flex-shrink", "flex-grow" e "flex-basis".

Item A = flex: 1 0 150px;
Item B = flex: 0 0 150px;
Item C = flex: 1 0 150px;

Item A
Item B
Item C

Obs.: É uma orientação da W3c que a propriedade simplificadora "flex" seja utilizada preferencialmente no lugar das três declarações.

Outras simplificações:

flex: 0 1 auto => flex: initial

flex: 0 0 auto => flex: none

flex: 1 1 auto => flex: auto
