Tamanho base dos itens

Flex-basis

Utilizamos a propriedade "flex-basis" para controlar o tamanho dos itens de um conteiner.

Por padrão a propriedade flex-basis sempre vem com o valor "auto" especificado. Isso significa que o tamanho dos itens é ditada de acordo com o tamanho do seu conteudo. Vamos usar conteúdos com diferentes tamanhos no exemplo abaixo.

Obs.: Para o exemplo abaixo usamos "flex-flow: row nowrap" para o conteiner.

Item AAAAA
Item B
Item CC
Item D
Item EEEEEE
Item F
Item GGGG

Flex-basis: 200px

Ao utilizarmos um valor para flex-basis como o de 200px por exemplo, todos os itens do conteiner terão esse tamanho.

Obs.: Lembre-se que, apesar do valor do item estar especificado como 200px, ele pode diminuir para se adaptar se o conteiner pai diminuir.
